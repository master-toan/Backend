# 📘 Khái niệm cơ bản về Request và Response trong Backend

## 1. Request (req)
- Là thông tin mà **client gửi lên server**.

## 2. Response (res)
- Là dữ liệu mà **server trả về cho client** sau khi xử lý.

## 3. Tóm tắt
- **Request (req)**: thông tin đi **vào server** từ client.
- **Response (res)**: dữ liệu đi **ra khỏi server** từ server về client.

👉 Đây là nền tảng cơ bản nhất để hiểu cách giao tiếp giữa client và server trong lập trình backend với Node.js và Express.
