# 📘 HTTP Methods cơ bản trong Backend

## 1. GET
- Dùng để **lấy dữ liệu** từ server.
- Ví dụ: lấy danh sách người dùng từ `/users`.

## 2. POST
- Dùng để **gửi dữ liệu mới** lên server.
- Ví dụ: tạo người dùng mới với dữ liệu `{ "username": "nikola" }`.

## 3. PUT
- Dùng để **cập nhật dữ liệu** đã có trên server.
- Ví dụ: cập nhật thông tin người dùng tại `/users/1`.

## 4. DELETE
- Dùng để **xóa dữ liệu** trên server.
- Ví dụ: xóa người dùng tại `/users/1`.

---

## Tóm tắt
- **GET** → lấy dữ liệu.  
- **POST** → thêm dữ liệu mới.  
- **PUT** → cập nhật dữ liệu.  
- **DELETE** → xóa dữ liệu.  

👉 Đây là 4 hành động cơ bản nhất để xây dựng API chuẩn RESTful trong backend với Node.js và Express.
